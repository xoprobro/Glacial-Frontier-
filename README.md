# Glacial-Frontier-
This is a game about surviving in the wilderness of Glacier National Park (GNP) 

Glacial Frontier will be a first person wilderness survival game in Montana GNP.

Graphics: 

1: This game will be in a low-poly design inspired by the landmarks in the National Park

2: This game will also include sunsets, fog, and cold weather.

Weather System:

1: Clear days where there will be more visibility for the player

2: Fog for reduced sight to make it difficult for the player

3: Snow making it harder for the player to traverse 

4: Storms making enviornments worse for the player

Hotkeys:

Action Key: Mouse 1 

Move Forward: W

Move Backward: S

Move Left: A

Move RIght: D 

Jump: Spacebar

Sprinting (if considered later on): Left Shift

Interact/Pick up: E
